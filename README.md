# Number Guessing Game 🎯

An interactive Java program where users try to guess a number with feedback.

## Features
- User input handling
- Conditional logic for checking guesses
- Simple interactive gameplay

## Technologies Used
- Java

## How to Run
1. Run the program
2. Enter your guess
3. Check if you are correct

## Example Output
Guess a number between 1 and 10:  
7  
Correct! You guessed it.
